How to test AI Chat:

1. Open the page in browser:
   http://localhost:3001/pages/spec-viewer.html?id=iAzaUwtSW3qvcW87lICL

2. Click on the "AI Chat" tab (should be available because overview is approved)

3. The system will try to upload the spec to OpenAI automatically

4. If there is a valid API key:
   - You will see "Initializing AI assistant..." message
   - Then you can ask questions

5. If there is no valid API key:
   - You will see an error message
   - You will need to add an API key in .env
