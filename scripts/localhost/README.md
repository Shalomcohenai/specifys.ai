# 🚀 Localhost Development

Quick setup for running Specifys.ai locally.

## Quick Start

```bash
./start-localhost.sh
```

This will start both frontend (Jekyll) and backend (Node.js) servers.

## Access

- **Frontend:** http://localhost:4000
- **Backend:** http://localhost:10000

## Testing

```bash
./test-localhost.sh
```

## Full Documentation

See [LOCALHOST-SETUP.md](./LOCALHOST-SETUP.md) for complete setup guide (in Hebrew).

## Requirements

- Node.js 18+
- Ruby 2.7+
- Bundler
- Backend `.env` file configured

## Files

- `start-localhost.sh` - Start both servers
- `test-localhost.sh` - Test that everything works
- `LOCALHOST-SETUP.md` - Complete setup guide (Hebrew)
