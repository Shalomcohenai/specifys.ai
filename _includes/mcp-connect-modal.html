Connect MCP

Use one API key per account. Cursor or Claude Desktop can then read and update your specs in real time.

Requirement: Node.js must be installed on your machine to use MCP (Cursor and Claude run the Specifys server via npx).

Setup

In Cursor open Settings → MCP. Add a server and paste the JSON below (no path needed — it uses the specifys-mcp-server package). Save and restart Cursor.

Open Claude Desktop config (e.g. ~/Library/Application Support/Claude/claude_desktop_config.json). Add the Specifys server from the JSON below; set env.SPECIFYS_API_KEY (and optionally SPECIFYS_API_BASE_URL). Restart Claude Desktop.
